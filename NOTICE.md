This repository is a curated public documentation repository for Orina Protocol.

## Repository Scope

The repository contains:

- Public-facing protocol documentation.
- Runtime user and reference documentation.
- Internal audit summaries suitable for public release.
- Formal verification summaries.
- Curated formal artifacts and bytecode files used for verification reference.
- Official public links and source inventory material.

The repository intentionally excludes private deployment runbooks, environment files, operator-only verification procedures, raw secrets, private keys, JWT signing material, service-role keys, generated API keys, local logs, and internal planning notes.
